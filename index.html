Projects

MKE Flavors

MKE Flavors updates daily with flavor of the day information that is agrregated from Milwaukee area frozen custard stands. It uses Google's geolocation API to find and show the locations closest to you. I created a REST API that is used for adding and retrieving data. This site was built using Angular, JavaScript, Node.js, Express and Mongo.

Live Site Github
Wisconsin Ticket Finder

Wisconsin Ticket Finder uses the StubHub API to display a list of upcoming events, along with information on ticket prices and availability. This site was built using React.

Live Site Github
MKE Flavors Alexa Skill

This skill was built using AWS Lambda and connects to the REST API that was created for my MKE Flavors app. It uses geolocation to find frozen custard locations near you and it can also be asked for flavors at a specific location. This skill has been submitted to Amazon for certification and I'll post a link as soon as it is available!
